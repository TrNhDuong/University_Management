Detailed Description

Các hàm tiện ích và lớp hỗ trợ cho hệ thống quản lý cơ sở dữ liệu.

Bao gồm các hàm để thêm, xóa, thay thế dữ liệu trong vector, hiển thị danh sách, kiểm tra lựa chọn hợp lệ và các hàm hỗ trợ khác.
