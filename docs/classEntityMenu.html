Detailed Description

Lớp đại diện cho menu thực thể trong hệ thống quản lý cơ sở dữ liệu.

Lớp này kế thừa từ Menu và cung cấp các tùy chọn cụ thể cho các thao tác trên thực thể, bao gồm sinh viên, giảng viên và khoa.

Note
Phương thức type sẽ trả về chuỗi "EntityMenu" để xác định loại menu này.
