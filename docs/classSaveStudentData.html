Detailed Description

Lớp đại diện cho máy lưu dữ liệu sinh viên.

Lớp này kế thừa từ ISaveData và triển khai phương thức execute để lưu dữ liệu sinh viên vào cơ sở dữ liệu.

Note
Lớp này sử dụng cơ sở dữ liệu đã được cung cấp để lưu trữ dữ liệu sinh viên.
Dữ liệu sẽ được lưu vào file "student.txt" trong thư mục hiện tại.
Dữ liệu sẽ được lưu theo định dạng: ID, Name, Birth, Mail, Class.
Nếu file không tồn tại, nó sẽ được tạo mới. Nếu đã tồn tại, dữ liệu sẽ được ghi đè.
