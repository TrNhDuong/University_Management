Detailed Description

Lớp đại diện cho menu chính trong hệ thống quản lý cơ sở dữ liệu.

Lớp này kế thừa từ Menu và cung cấp các tùy chọn cụ thể cho menu chính, bao gồm các thao tác trên cơ sở dữ liệu.

Note
Phương thức type sẽ trả về chuỗi "MainMenu" để xác định loại menu này.
